## 🎨 ロゴ画像（`downloads/logos/`）

| `likanon-circle-black_*` | **メイン**（GBP・Instagram・SNS）| 黒丸＋白リンク |

## 🚀 すぐに使うアクション

### ① GBPプロフィール写真をアップ（5分）
1. GBP管理画面 → 写真 → ロゴ
2. `downloads/logos/likanon-circle-black_2048.jpg` をアップ

### ② 制作実績を投稿（10分）
1. GBP管理画面 → 投稿を作成
2. `downloads/portfolio/portfolio-01-nicolon.jpg` をアップ
3. `GBP_OPERATION_TEMPLATES.md`の投稿6のテキストをコピペ
4. 「詳細」リンクで nicolon の記事URLを貼る

### ③ Instagramプロフィールに統一
1. Instagramアプリ → プロフィール編集
2. `downloads/logos/likanon-circle-black_1080.png` をアップ
