### 3. Google広告 小予算スタート（所要1〜2時間・1週間で反応）

**月3〜5万円の予算で、月3〜10件の問い合わせ獲得を狙う**のが現実的なライン。

#### 🎯 出稿手順（初心者向け）

**ステップ1：Googleアカウントで広告アカウント開設**
1. https://ads.google.com にアクセス
2. 「今すぐ開始」→ Googleアカウントでログイン
3. ビジネス情報入力（事業者名・URL）
4. **「エキスパートモードに切り替える」をクリック**（スマートキャンペーンは効果薄）

**ステップ2：最初のキャンペーン作成**
1. 「キャンペーン」→「新しいキャンペーン」
2. 目標：「リードの獲得」を選択
3. キャンペーンタイプ：「検索」を選択

**ステップ3：キーワード設定（コピペで使えます）**

最初に試すキーワード（10個でOK）：
```
ホームページ制作 安い
HP制作 個人事業主
ホームページ制作 5万円
ホームページ 月5名限定
HP 作成 小規模
個人事業主 HP 必要
ホームページ制作 福岡
ホームページ制作 リニューアル
ホームページ 集客
WEB制作 フリーランス
```

地域名がある場合（福岡なら）：
```
ホームページ制作 福岡
HP制作 福岡 個人事業主
福岡 ホームページ 安い
```

**ステップ4：広告文の作成**

見出し（30文字以内×3つ）：
- 「【月5名限定】HP制作5万円〜」
- 「個人事業主のためのWeb制作」
- 「30分無料相談・しつこい営業なし」

説明文（90文字以内×2つ）：
- 「個人事業主・小規模店舗専門。月5名限定で5万円〜の本格HP制作。SEO・スマホ対応・フォーム込み。」
- 「100件以上の実績。無料診断・無料相談から。LINE対応OK。中間マージンゼロで低価格×高品質。」

**ステップ5：予算設定**
- 1日の予算：**¥1,000〜¥1,500**から
- 月予算換算：3〜5万円
- 入札戦略：「クリック数の最大化」（最初の1ヶ月）

**ステップ6：1週間後の見直し**
1週間運用したら：
- クリック単価（CPC）が高すぎないか（200円以下が理想）
- どのキーワードで反応があるか
- 不要なキーワードを除外（「無料 ホームページ 作り方」など、購入意図のないクエリ）

## ❓ よくある不安と回答

**Q. 月3〜5万円の広告費は高くないか？**
→ HP制作1件が5万円〜10万円の利益なので、月1件成約で元が取れます。最初は赤字でも、データを取ることに価値があります。

**Q. 既存顧客にLINE送るのは申し訳ない**
→ 「強引な営業」ではなく「紹介してもらえたら嬉しい」のスタンスで送れば問題ありません。むしろ「教えてくれてありがとう」と返ってきます。

**Q. SNS投稿が続く自信がない**
→ 1日1投稿で十分。10日続けるとリーチが安定し始めます。完璧でなくていいので、まず投稿を続けることが重要。

**Q. 広告って怖い**
→ 1日¥1,000の設定なら、最悪3万円/月で止まります。毎日5分の確認だけで運用可能。1週間で「やってよかった」or「もう少しチューニングする」が判断できます。
