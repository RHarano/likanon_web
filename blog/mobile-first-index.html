モバイルファーストインデックス対応ガイド

2021年に完全移行したモバイルファーストインデックス。Googleはスマホ版のサイトを基準に順位を決定します。「PCで作って、スマホはおまけ」だった時代は終わりました。

モバイルファーストインデックスとは

Googleがサイトのモバイル版を主にクロール・評価する仕組み。PC版は補助的に扱われる。検索の70%以上がスマホ経由になった結果、Googleが2021年に完全切り替えた。

影響を受けるサイト

  • レスポンシブ対応していないサイト → 致命的
  • PCとスマホで別URL運用しているサイト → 注意
  • スマホ版にコンテンツを減らしているサイト → 評価低下

対応チェックリスト

  • レスポンシブWebデザインを採用
  • スマホでも全コンテンツが表示される
  • スマホで読みやすいフォント（16px以上）
  • タップ可能領域が十分（48px以上）
  • スマホでの表示速度（LCP 2.5秒以内）
  • 横スクロール発生なし
  • ポップアップが画面を埋め尽くさない

確認方法

  1. Search Console「モバイルユーザビリティ」レポート
  2. Google「モバイルフレンドリーテスト」（無料ツール）
  3. 実機（自分のスマホ）で全ページ確認
  4. Chrome DevToolsのスマホシミュレーション

WordPressでの対応

  • レスポンシブ対応のテーマを選ぶ（Cocoon・Lightning・SWELL等）
  • 「テーマカスタマイザー」でモバイル表示確認
  • AMPプラグインは2025年で非推奨に

まとめ

  • スマホ版が評価基準
  • レスポンシブ対応必須
  • PCとスマホで内容を変えない
  • 表示速度・タップ領域も重要
