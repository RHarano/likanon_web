個人ECサイト立ち上げ完全ガイド｜Shopify vs BASE vs STORES

個人事業主・ハンドメイド作家・小規模小売店向けのEC立ち上げガイド。プラットフォーム選定から運営の基礎まで、初期投資5万円以内で始める手順を解説します。

3つの主要プラットフォーム比較

Shopify

  • 料金：月33ドル〜
  • 機能：最強・拡張性も高い
  • 世界標準・越境ECも対応
  • 個人〜中堅事業者向け

BASE

  • 料金：初期費用無料・販売手数料3%＋40円/件＋6.6%（決済）
  • 機能：基本機能十分・日本特化
  • 個人ハンドメイド・少数販売向け

STORES

  • 料金：フリープラン有・月3,480円〜のスタンダード
  • 機能：直感的UI・デザイン豊富
  • 個人〜小規模事業者向け

どれを選ぶか

  • 本気で売上を伸ばしたい→Shopify
  • まず試したい・少数販売→BASE
  • デザイン重視・初心者→STORES
  • 越境EC視野→Shopify

立ち上げの7ステップ

  1. プラットフォーム選定
  2. 商品撮影（10商品でも30枚以上）
  3. 商品説明文の作成
  4. 決済方法の設定（クレカ・コンビニ・銀行）
  5. 配送設定（地域別送料）
  6. 特商法・プライバシーポリシー記載
  7. テスト購入

初期投資の目安

  • プラットフォーム月額：0〜5,000円
  • 商品撮影機材：1〜3万円（スマホ＋ライト）
  • 梱包資材：5,000円〜
  • 商品仕入れ：個人作家ならゼロ
  • 合計：1〜5万円

運営の継続ポイント

  • 商品の写真品質が売上の8割
  • 商品説明は悩み解決軸で書く
  • Instagram連動で集客
  • レビュー獲得を仕組み化
  • リピーター施策（メルマガ・LINE）

まとめ

  • 3プラットフォームから事業フェーズで選ぶ
  • 初期投資1〜5万円で十分
  • 商品写真の品質が最重要
  • Instagram連動が定石
